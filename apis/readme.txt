# Instrucciones para ejecutar las pruebas de API con Cypress

1. Abre una terminal en la carpeta "apis".
2. Instala las dependencias:
   npm install
3. Abre Cypress:
   npm run cypress:open
4. Ejecuta la prueba:
   Selecciona el archivo "demoblaze_api.cy.js" en la interfaz de Cypress y ejecuta la prueba.

Alternativamente, puedes ejecutar la prueba en modo headless:
   npm run cypress:run

Las pruebas automatizan los casos de registro y login en la API de Demoblaze.
